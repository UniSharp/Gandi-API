Zend_Service_Gandi

Zend Framework Gandi API Service extension.

Zend_Service_Gandi provides a simple PHP library for the Gandi API.

Gandi API documentation: http://doc.rpc.gandi.net

Dependencies

  • PHP 5.2.4+
  • Zend Framework: http://www.framework.zend.com/downloads/latest#ZF1
  • Gandi account: https://www.gandi.net

How to use

<?php
require_once 'Zend/Service/Gandi.php';

try {
    $gandi = new Zend_Service_Gandi('your_api_key');
    // Get VM infos
    print_r($gandi->getVmInfo('your_vm_id'));
} catch (Exception $e) {
    echo $e->getMessage();
}

Available Methods

  • Hosting
    • getAccountInfo(): Get Gandi Hosting account info
    • getVmList(): Get VM list
    • getVmInfo($id): Get VM info by VM Id filter
    • getGraph($vmId, $target, $deviceNumber): Return graph URL
  • PASS
    • getPassInfo($id): Get Gandi PASS info
